Threadwick Studio — redesign mockups

Static, browser-viewable mockups of every locked screen from the Studio redesign. They share one stylesheet (shared.css) in Threadwick's warm Brick & Ecru light palette. Gray blocks are image placeholders. The full rationale lives in spec.md.

These are visual references for hand-off, not production markup — inline styles and fixed sizes keep each file self-contained. Treat them as the source of truth for layout and hierarchy, not for code structure.
