## Context

Threadwick is built primarily by autonomous agents. Work must be recorded, documented, and closed
with no external tracker and no human in the loop. This task introduces the repo-native `work/`
ledger as the source of truth, with GitHub as a read-only mirror, before Phase 6 starts.

## Scope

In: the work/ schema + lifecycle, scripts/work.ts (check/index/next/list/new), the hard CI gate,
CONTRIBUTING + MIGRATION + CLAUDE.md updates, and seed tasks for Phase 6.
Out: the one-way GitHub Projects mirror Action and the daily stale-sweep (follow-up tasks).

## Acceptance

- [x] work/ lifecycle documented in work/README.md and CONTRIBUTING.md

- [x] CI runs `pnpm run work check` as a hard, blocking job
